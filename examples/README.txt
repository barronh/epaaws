epaaws Examples
==============

epaaws has the ability to download, open, and plot EPA modeling platform products from AWS.


The examples assume that you have installed epaaws:

.. code-block:: bash

    python -m pip install epaaws


After that, you can copy or download the code to run it.

If something is unclear, feel free to open an `issue <https://github.com/barronh/epaaws/issues>`_.
